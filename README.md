# Tunnelmagasin-Groedalen-kraftverk

Dette GitHub-arkivet inneholder python-kode og CSV-filer benyttet i en bacheloroppgave for tre studenter ved NTNU. Det er utført beregninger for hydrologiske data, analyse av priser i kraftmarkedet, beregninger av produskjon og inntekt, samt visualisering av tunnelmagasin. Oppgaven omhandler prosjektering av magasin i tunnell for å se om det øker lønnsomheten til Grøddalen småkraftverk. Denne siden linkes til i oppgaven som et vedlegg. De vedlagte CSV-filene er anvendt i kodene.
